A simple web-based application that tracks tree plantation records and predicts tree survival rates using Machine Learning.

## Features
- Record tree plantation details
- Predict survival rate using AI
- Learn from actual survival data
- Store records using SQLite
- Simple and clean UI

## Tech Stack
- Python
- Flask
- SQLite
- Machine Learning (Random Forest)
- HTML/CSS

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Initialize database:
   python init_db.py

3. Run the app:
   python app.py

4. Open browser:
   http://127.0.0.1:5000/
